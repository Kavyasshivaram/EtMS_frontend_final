# EtMS Frontend (React)

This is the premium React.js frontend for the Education & Training Management System (EtMS).

## 🚀 Quick Start
1. `npm install`
2. `npm run dev`

## 🔗 Related Repositories
- **Backend (API)**: [Etms-backend](https://github.com/vinni868/Etms-backend)

## ✨ Premium UI Features
- **Modern Dashboard**: Glassmorphic and responsive design for all user roles.
- **Universal ID Display**: Standardized IDs (e.g., STU-2026-0001) visible across all portals.
- **Management Portals**: Full control for Admins and Super Admins over the entire userbase.
